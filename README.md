# STManagement-System


A full-stack web application designed to simplify and digitalize student data management. This system provides an efficient, secure, and user-friendly platform for managing students, courses, and related activities in real time.

🚀 Live Demo

🔗 https://lnkd.in/djZSamBS

💻 GitHub Repository

🔗 https://lnkd.in/dacUMUmb

📌 Features

🔐 Secure Authentication using Firebase Authentication

➕ Add Students & Courses

✏️ Edit Records

❌ Delete Records

📋 Audit Logs to track all system activities

⏰ Reminder System for important deadlines

📱 Responsive Design for mobile and desktop

⚡ Real-time Data Updates with Firebase Realtime Database

🛠️ Tech Stack
Layer	Technology
Frontend	React.js
Backend	Firebase
Database	Firebase Realtime Database
Authentication	Firebase Authentication
Hosting	Firebase Hosting
Tools	Node.js, Git, GitHub

⚙️ Installation & Setup

Follow these steps to run the project locally:

1. Clone the repository
git clone https://github.com/your-username/STManagement-System.git
2. Navigate to project folder
cd STManagement-System
3. Install dependencies
npm install
4. Setup Firebase

Create a project in Firebase

Enable Authentication and Realtime Database

Add your Firebase config in the project

5. Run the app
npm start
🔑 Environment Configuration

Create a .env file and add your Firebase credentials:

REACT_APP_API_KEY=your_api_key
REACT_APP_AUTH_DOMAIN=your_auth_domain
REACT_APP_DATABASE_URL=your_database_url
REACT_APP_PROJECT_ID=your_project_id
REACT_APP_STORAGE_BUCKET=your_storage_bucket
REACT_APP_MESSAGING_SENDER_ID=your_sender_id
REACT_APP_APP_ID=your_app_id
